# Mein Crypto Token

Ein einfacher ERC-20 Token auf Polygon mit Mint und Burn Funktionen.

## Features
- ✅ ERC-20 Standard
- ✅ Mint Funktion (nur Owner)
- ✅ Burn Funktion (jeder kann eigene Tokens verbrennen)
- ✅ Max Supply: 10 Millionen Tokens
